# Soldering Fume Extractor
Its pretty simple, but VERY useful

## Why
The main reason im making this, is for my health.
When im soldering it seems like the smoke tries its best to come towards me.

## Features
- It can Tilt
- It Extracts Fumes
- Speed Control
- Under $30(depending on shipping)

## BOM
- Female Plug 12V DC
- 12V Plug adapter
- 12V DC Motor Speed Controller Module
- PC Fan
- 3mm Sponge Foam
- 10mm Activated Carbon Foam Sponge
- Rocker Switch
  
- Soldering-Fume-Extractor.step
